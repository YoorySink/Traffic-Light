Komponen Utama:
- Arduino Uno: Sebagai otak/pengendali utama skema lampu.
- 12 LED: 4 Merah, 4 Kuning, dan 4 Hijau.
- Resistor: Digunakan untuk membatasi arus listrik agar LED tidak putus (terbakar).
- Jumper Wires: Kabel penghubung antar komponen.
