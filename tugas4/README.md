## Daftar Komponen
* **Arduino Uno** - Sebagai mikrokontroler utama.
* **12 LED** - Terdiri dari Merah, Kuning, dan Hijau.
* **Resistor** - Untuk membatasi arus LED.
* **Kabel Jumper** - Untuk koneksi antar komponen.

## Pin Mapping
1. Sisi Utara: Pin 2, 3, 4
2. Sisi Timur: Pin 5, 6, 7
3. Sisi Selatan: Pin 8, 9, 10
4. Sisi Barat: Pin 11, 12, 13
