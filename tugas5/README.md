## Daftar Komponen
* **Arduino Uno** - Sebagai mikrokontroler utama.
* **9 LED** - Terdiri dari Merah, Kuning, dan Hijau.
* **3 Resistor** - Untuk membatasi arus LED.
* **Kabel Jumper** - Untuk koneksi antar komponen.
* **Push button** - tombol lampu pejalan kaki

## Pin Mapping
1. led merah = 2, 5, 8
2. led kuning = 3
3. led hijau = 4, 7, 10
4. Button = 11, 12
